Configuration

Votre cle reste dans votre navigateur. Elle n'est ni stockee, ni journalisee, ni envoyee ailleurs qu'a l'API Mistral.

Confidentialite

Aucune donnee n'est stockee. Les fichiers audio, transcripts et comptes rendus existent uniquement le temps de la session, dans votre navigateur. Telechargez vos resultats pour les conserver. L'audio est transmis a l'API Mistral pour la transcription. Ne traitez pas de donnees sensibles sans validation interne.

Compte rendu de reunion depuis un audio

Transformez un enregistrement de reunion en transcript, puis, si vous le souhaitez, en compte rendu professionnel telechargeable.

Pour commencer, saisissez votre cle API Mistral dans la barre laterale. Tant que la cle n'est pas fournie, les etapes restent desactivees.

1. Deposez votre enregistrement

Formats audio acceptes : mp3, wav, m4a, flac, ogg. (Version web : audio uniquement, pas de conversion video.)
